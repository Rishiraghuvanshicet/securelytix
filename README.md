## 🚀 Setup Instructions

1. **Clone the Repository**

```bash
git clone https://github.com/Rishiraghuvanshicet/securelytix.git
cd securelytix
```

2. Install Dependencies

```bash
npm install
Run the Development Server
```
3. Run the Development Server

```bash
npm run dev
```

4.Build for Production

```bash
npm run build
```
